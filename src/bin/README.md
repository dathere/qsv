# qsv Claude Agent SDK Helpers

## qsv-skill-gen
**Generate Agent Skills from qsv command USAGE text**  
This tool parses USAGE text from qsv commands and generates Agent Skill
definitions in JSON format for use with the Claude Agent SDK. Uses qsv-docopt Parser for robust USAGE text parsing.

See [Claude Agent Skills README](../../.claude/skills/README.md) for more info.
